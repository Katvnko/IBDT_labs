# Лабораторна робота №4  
## Перетворення шістнадцяткові (hex) значення у цілі беззнакові числа (int)

### Мета роботи
набуття практичних навичок з обробки та представлення даних у пам'яті комп'ютера. Студенти повинні навчитися конвертувати шістнадцяткові (hex) значення у цілі беззнакові числа, враховуючи порядок байтів (little-endian або big-endian). Це дозволить зрозуміти, як дані, що перевищують розмір одного байта, зберігаються та інтерпретуються в різних архітектурах.

### Теоретичні відомості
У комп’ютерних системах дані зберігаються у вигляді байтів (8 біт).
Числа, що перевищують розмір одного байта, зберігаються як послідовність байтів.
Порядок зберігання байтів залежить від архітектури:

- **Little-endian** — молодший байт зберігається першим
- **Big-endian** — старший байт зберігається першим

Коректна інтерпретація таких даних потребує правильного врахування порядку байтів
під час перетворень.

### Функціональність бібліотеки
Реалізовано бібліотеку, яка підтримує:

- перетворення HEX до Little-endian integer
- перетворення HEX до Big-endian integer
- перетворення Little-endian integer до HEX
- перетворення Big-endian integer до HEX

### Реалізація
Мова програмування: **Python**

Для перетворень використовуються стандартні механізми роботи з байтами,
що дозволяє коректно обробляти значення будь-якої довжини.

### Тестування
Реалізація перевірена на тестових векторах, наведених у завданні,
включно зі значеннями довжиною байтів.
