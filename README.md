# Шифрування та дешифрування AES-128 (Advanced level)

Цей репозиторій містить реалізацію алгоритму **AES-128 (Advanced Encryption Standard)** у межах лабораторної роботи №1.  
Реалізація демонструє **шифрування та дешифрування** текстових даних із використанням алгоритму AES.

---

## Реалізований алгоритм

- **Алгоритм:** AES
- **Довжина ключа:** 128 біт
- **Режим роботи:** ECB
- **Паддінг:** PKCS7
- **Формат зашифрованих даних:** Base64

Реалізація розділена на два логічні модулі:
- `encrypt.py` — шифрування відкритого тексту
- `decrypt.py` — дешифрування зашифрованого тексту

---

## Вимоги

- Python 3.8 або новіший
- Бібліотека: `pycryptodome`

### Встановлення залежностей
```bash
pip install pycryptodome
```
Коректність реалізації підтверджено шляхом успішного відновлення початкового тексту після дешифрування результату шифрування.

---
### Перевірка

Для додаткової перевірки
echo -n "Try to do 1 lab" | \
openssl enc -aes-128-ecb -K 4b61746572796e61566173686368656e -base64

echo "<вставити код після 1 команди>" | \
openssl enc -aes-128-ecb -d -K 4b61746572796e61566173686368656e -base64
